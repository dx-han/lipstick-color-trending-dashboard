# lipstick-color-trending-insight-dashboard

Color is a key factor in attracting customers to make a purchase. I build a trending insight dashboard focusing on color. It can help R&D and MKT teams to find the next potential color on time.

# Technology

Shiny, JavaScript, R, html, sass

# Get started

Go to the root folder and execute the following commands using your favourite command line.
```
docker build -t lipstick .
docker run -it --name lipstick -p 3839:3838 lipstick
```

Open your favourite browser and go to `127.0.0.1:3838/lipstick/app`.

# Announcement

Any data in this demo have been desentisized. It is useless to get any real insight which can be used in the real world.
